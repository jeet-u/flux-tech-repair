Weekly series focused on professional microsoldering: shorts, communication lines, Face ID, CPU work, reballing, and dead device recovery. Fresh technical content every week.

## About Microsoldering Weekly

Microsoldering Weekly is a featured series designed for technicians and engineers who work with logic board repair and microelectronics. It is perfect for publishing regularly updated, hands-on content related to professional microsoldering, such as:

- Weekly microsoldering case studies
- Logic board diagnostics notes
- Learning progress in microelectronics
- Repair workflow and project updates

## Microsoldering Weekly Configuration

Configure in `config/site.yaml`:

```yaml
featuredSeries:
  categoryName: weekly       # Category name
  label: Microsoldering Weekly          # Display label
  fullName: Professional Microsoldering Weekly  # Full name
  description: Weekly insights into logic board repair, diagnostics, and microsoldering techniques.
  cover: /img/weekly_header.webp
  enabled: true             # Set to false to disable
```

## Microsoldering Weekly Features

1. **Dedicated Page** - Microsoldering Weekly has its own `/weekly` page
2. **Homepage Display** - The latest microsoldering issue is pinned at the top of the homepage
3. **Separate List** - Weekly issues do not appear in the regular article list
4. **Series Navigation** - Easy navigation between previous and next microsoldering issues
